# Portfolio Site

A personal portfolio website built with **React + Vite**, designed as my home for projects, experiments, and interactive visual work.

This repo is the foundation for a modern portfolio experience with smooth navigation, fast performance, and room for playful Three.js-powered components.

---

## Tech Stack

- **React** (frontend UI)
- **Vite** (dev + build tooling)
- **React Router** (multi-page navigation)
- **Three.js + React Three Fiber** (3D + interactive visuals)
- **Netlify** (deployment)

Dependencies include `react-router-dom`, `three`, `@react-three/fiber`, and related tooling.
